タイトルを指定して、Kensaのインスタンスを生成します。

```typescript
import Kensa from 'kensajs';

const ks = Kensa('テストタイトル');
```

ksには、以下のメソッドがあります。

- [test](#test)
- [testAsync](#testAsync)
- [before](#before)
- [after](#after)
- [beforeEach](#beforeEach)
- [afterEach](#afterEach)
